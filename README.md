## 🌟 Overview

**CareerForge-Pro** is a premium, full-stack AI-driven platform designed to empower professionals to build high-impact, ATS-optimized resumes in minutes. By merging advanced AI content generation with designer-grade templates, we provide the tools necessary to bridge the gap between world-class talent and top-tier opportunities.

## ✨ Key Features

### 🤖 Intelligent AI Content Strategy
*   **Precision Rewriting:** Transform basic bullet points into powerful, results-oriented achievement statements using industry-leading AI models.
*   **Contextual Optimization:** Automatically align your resume content with specific Job Descriptions (JD) to maximize relevance.
*   **ATS Scoring Engine:** Real-time feedback on how well your resume matches target industry standards.

### 🎨 Designer-Grade Templates
*   **Premium Selection:** Access a curated library of templates including `Minimalist Elite`, `Executive Professional`, `High-Impact Corporate`, and `Modern Tech`.
*   **Real-Time Rendering:** Experience instant visual feedback with our high-fidelity "What You See Is What You Get" (WYSIWYG) editor.

### 📄 Bulletproof Document Export
*   **Crystal-Clear PDF Engine:** Proprietary "Direct Transfer" technology ensures sharp, vector-quality PDF exports at 2x resolution.
*   **Structural Integrity:** Optimized for parsing by major Applicant Tracking Systems (ATS) like Workday, Greenhouse, and Lever.

### 💳 Pro Account & Subscriptions
*   **Integrated Payments:** Professional Stripe integration for seamless account upgrades.
*   **Premium Features:** Unlock AI summary generation, advanced ATS analysis, and exclusive templates.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   MongoDB Atlas Account
*   Stripe API Keys (for payments)

### Installation

1.  **Clone the Repository**
    ```bash
    git clone https://github.com/AbhinavM-1/AI-RESUME-BUILD-ARCHITECH.git
    cd AI-RESUME-BUILD-ARCHITECH
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # Also install backend dependencies if separated
    cd server && npm install && cd ..
    ```

3.  **Environment Configuration**
    Create a `.env` file in the `server` directory:
    ```env
    PORT=5000
    MONGODB_URI=your_mongodb_connection_string
    JWT_SECRET=your_jwt_secret
    STRIPE_SECRET_KEY=your_stripe_key
    OPENAI_API_KEY=your_openai_key
    ```

4.  **Run Locally**
    ```bash
    # Run both client and server concurrently
    npm run dev
    ```
