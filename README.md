# AI Resume Build Architect

AI Resume Build Architect is a full-stack AI-powered resume builder web application designed to help users create professional, ATS-friendly resumes with smart content generation, modern templates, live preview, and PDF export functionality.

---

## Features

- AI-powered resume content generation
- ATS-friendly resume templates
- Smart resume editing
- Live resume preview
- Professional resume builder
- Add / Edit Personal Information
- Add / Edit Experience
- Add / Edit Education
- Add / Edit Skills
- Export resume as PDF
- Share resume with link
- Clean and modern UI

---

## Tech Stack

### Frontend
- React.js
- React Router DOM
- CSS3
- React Icons

### Backend
- Node.js
- Express.js
- MongoDB

### Tools & Libraries
- jsPDF
- html2canvas
