# 🚀 CareerForge-Pro: The Ultimate AI-Powered Resume Architect

**CareerForge-Pro** is a premium, full-stack AI-driven platform designed to empower professionals to build high-impact, ATS-optimized resumes in minutes. Combining cutting-edge AI generation with designer-grade templates, it bridge the gap between talent and opportunity.

---

## ✨ Key Features

### 🤖 AI-Powered Content Strategy
- **Smart Rewriting**: Leverage AI to transform simple bullet points into high-impact professional achievement statements.
*Note: AI features currently use optimized mock patterns due to quota limitations.*

### 🎨 Designer-Grade Templates
- **Premium Selection**: Access curated templates including `Modern Centered`, `Executive Professional`, `Minimalist Elite`, and `High-Impact Corporate`.
- **Real-Time Preview**: Experience instant visual feedback as you build your resume.

### 📥 Bulletproof PDF Export
- **Crystal-Clear Rendering**: Proprietary 2.2x scale "Direct Transfer" engine for sharp, print-ready documents.
- **Precision Layouts**: Guaranteed structural integrity for ATS (Applicant Tracking Systems) compatibility.

### 🔐 Secure & Social Authentication
- **Universal Login**: Seamless access via GitHub, Google, Apple, or traditional secure email authentication.
- **Persistent Storage**: Save your progress and multiple resume versions securely in the cloud.

---

## 🛠️ Technology Stack

### **Frontend Excellence**
- **Core**: React.js (Hooks & Functional Components)
- **Styling**: Vanilla CSS3 with Modern CSS Variables for high-performance UI.
- **Export Engine**: Optimized implementation of `html2canvas` and `jsPDF`.
- **Icons**: Curated sets from `react-icons` (FontAwesome, Lucide, FlatColorIcons).

### **Backend Stability**
- **Engine**: Node.js & Express.js
- **Database**: MongoDB (Atlas)
- **Authentication**: JWT-based Secure Auth & Social OAuth integration.

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/AbhinavM-1/AI-RESUME-BUILD-ARCHITECH.git
cd AI-RESUME-BUILD-ARCHITECH
```

### 2. Install Dependencies
```bash
# In the root directory
npm install
```

### 3. Configure Environment Variables
Create a `.env` file in the root and add:
```env
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
OPENAI_API_KEY=your_key
```

### 4. Run the Platform
```bash
npm start
```
